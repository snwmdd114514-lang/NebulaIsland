# 🌌 NebulaIsland

**NebulaIsland** 是一个 Roblox 灵动岛 UI 系统。  
模拟 iOS Dynamic Island，支持通知堆叠、控制中心、动画交互等。

---

## ✨ 功能

- 🍎 Apple Dynamic Island UI
- 📢 通知堆叠系统
- 🎚 控制中心模块
- 🔘 iOS 风格 Toggle Switch
- 🎞 丝滑 Tween 动画
- 📦 灵动岛 API 调用
- 🔔 自定义通知

---

## 📦 安装

直接在脚本中加载：

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/snwmdd114514-lang/NebulaIsland/main/NebulaIsland.lua"))()
```
